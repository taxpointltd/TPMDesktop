# **App Name**: TransactWise

## Core Features:

- User Authentication: Secure user login and registration to manage access to company data using Firebase Authentication.
- Company Management: Create, edit, and manage multiple company profiles within the app. If the company exists, allow users to select from an existing companies to affiliate to
- Data Import: Import vendor, customer, and chart of accounts data from Excel sheets, populating necessary database tables using batch processing.
- Transaction Upload: Allow users to upload transaction data from Excel sheets.
- Automated Transaction Matching: Use Gemini AI to automatically match uploaded transactions with vendors, customers, and chart of accounts. The AI is a tool that helps you connect existing companies, chart of accounts, customers and vendors.
- Review and Edit Matches: Enable users to review, edit, and confirm the matches generated by the AI before final export.
- Transaction Export: Export reviewed and matched transaction data back into an Excel format.

## Style Guidelines:

- Primary color: Navy blue (#2E3192) for trustworthiness and professionalism.
- Background color: Light gray (#F0F2F5), a very desaturated hue near navy, providing a clean backdrop.
- Accent color: Teal (#008080) to highlight important actions and interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern and neutral aesthetic suitable for both headlines and body text.
- Use minimalist icons to represent vendors, customers, accounts, and transaction types.
- A clean, data-dense layout with clear sections for each step of the transaction matching process.
- Subtle transitions and animations to provide feedback on actions and guide the user through the workflow.
